Tentang Kami Dev Web

Tentang Dev Web – Web Tutorial Indonesia

Dev Web.com merupakan web yang berisikan tutorial tentang Script membuat program atau aplikasi, yang berfokus pada pembaca indonesia yang mempunyai minat dalam hal pemrograman.

Dev Web.com sekarang ini masih baru memiliki konten tutorial pemrograman tentang membuat web, membuat blog, tips trik HTML, tips trik CSS dan lainya. Pada saat ini Dev Web.com masih berfokus untuk memberikan tutorial pemrograman web yang berupa HTML, CSS, JS, PHP dan kedepannya akan disajikan tutorial pemrograman yang lebih luas.

Konten Tutorial Dev Web.com ditulis secara detail, sangat sederhana, mudah untuk dipahami bagi pemula, memiliki langkah-langkah yang lengkap dan beberapa tutorial telah dilengkapi gambar di tiap langkah-nya. Tujuan lain dari pembuatan web DevWeb.com adalah untuk media sharing pengalaman kami sebagai programmer, untuk media saling berbagi mengenai tips dan trik dalam dunia pemrograman, dan juga sebagai media penampung segala kritik dan saran yang membangun agar kedepan-nya website ini supaya lebih bermanfaat.

Dev Web.com baru berdiri pada bulan awal bulan November tahun 2015 dalam memenuhi tugas sekolah tentang web statis, kami akan selalu berupaya untuk memberikan tutorial yang selalu up-to-date, sehingga konten akan selalu bisa digunakan sebagai bahan belajar dan bahan referensi. Meskipun di share secara gratis, diharapkan konten tidak di salin tanpa izin dari kami.

Meskipun terbilang baru, kami optimis jika web yang bersifat tutorial akan selalu memberikan manfaat dengan waktu yang lama, dan kami bercita-cita menjadikan situs Dev Web.com menjadi web tutorial yang selalu bermanfaat dalam menyajikan tutorial pemrograman berbahasa indonesia.

Dev Web.com memberikan halaman khusus untuk mencoba kode HTML, CSS dan Javascript secara online di halaman Editor!, Website ini masih dalam tahap pengembangan, maka sekali lagi segala koreksi dan kritikan yang positif akan selalu kami nantikan dari para pembaca. Jika terdapat pertanyaan atau ingin menghubungi kami, silahkan mengunjungi halaman Kontak Kami.

Yang terakhir kami ucapkan terimakasih sudah berkunjung di website sederhana ini dan semoga bermanfaat.
